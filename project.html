Minor Project

Weather_App

This web-based Weather App allows users to input their location and view real-time weather information such as temperature, description, humidity, and wind speed. The application fetches weather data through a JavaScript-powered API call, and the results are dynamically displayed on the page.

Voise-To-Text

This is a simple Text-To-Voice application that allows users to convert spoken words to text. It uses HTML, CSS, and JavaScript to provide a user-friendly interface for voice-to-text conversion.

Text-to-Voice

This Text-to-Voice web application allows users to input text into a text box and have it converted to speech. The app uses JavaScript functionality to read the text aloud. It provides an easy and interactive way to hear text content spoken in real-time.

Instagram-Profile-View

This HTML file creates a simple web page that allows users to generate a link to an Instagram profile by entering a username. It includes an input field where users can type in the Instagram username, and upon clicking a button, the page generates a clickable link to that profile.

ROCK-PAPER-SCISSOR-GAME

This is a simple Rock Paper Scissors game implemented using HTML, CSS, and JavaScript. Users can play the game by selecting one of the three options: Rock, Paper, or Scissors. The computer will randomly choose its move, and the game will display the result.

Password-Generate

This project is a Password Generator web application that generates a random password based on the user's input, including their first name, last name, and a number.

QR-Code-Generator

This HTML code creates a simple web page that allows users to generate QR codes by entering text.


Github-Profile-Viewwer

This HTML document is designed to create a simple GitHub Profile Viewer web tool. It allows users to view a specific GitHub profile by entering a GitHub username.

Character Count

This is a simple web application that counts the number of characters typed by the user. It was built using HTML, CSS, and JavaScript. The goal of this project is to demonstrate the ability to build an interactive webpage that updates in real-time as the user types in a text area.
